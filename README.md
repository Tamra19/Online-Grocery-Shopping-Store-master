# Online-Grocery-Shopping-Store-master
ShopFreshly is a Django-based web application designed for managing grocery orders online. 
It allows users to browse products, add to cart, and place orders seamlessly. 
This project was built to demonstrate full-stack development with Python and Django.

## Features
- User authentication
- Product listing
- Cart and checkout system
- Admin dashboard
- SQLite supported

## Tech Stack
- **Backend:** Python, Django
- **Frontend:** HTML, CSS, JS, Bootstrap
- **Database:** SQLite (Default django database)
- **Tools:** Git, VS Code

## Installation
- Clone the repository
- Create and activate virtual environment
- Install dependencies from requirements.txt
- Apply migrations
- Run the server

## Usage 
- Open your browser and go to http://127.0.0.1:8000/
- Create a new account or login as admin
- Start exploring the features (add items, manage orders, etc.)
